D2W Diff — 使い方

FigmaデザインをWebページにオーバーレイして比較するツール

セットアップ
  1. 1
    Figma PAT を取得
    Figma → アカウント設定 → Personal access tokens からトークンを発行。figd_ で始まる文字列です。
  2. 2
    PAT を入力して保存
    初回起動時のセットアップ画面にトークンを貼り付け、「保存して続ける」をクリック。ブラウザを閉じても保存したい場合は「ブラウザ終了後も保存する」にチェック。
  3. 3
    Figma URL を入力
    FigmaのデザインURLをコピーして貼り付け。?node-id= パラメータ付きのURLが必要です。最大5件まで保存できます。
  4. 4
    「オーバーレイを読み込む」をクリック
    現在開いているWebページにFigmaデザインが半透明で重なります。
画像アップロードモード
  1. 1
    「画像アップロード」タブを選択
    Figma APIなしで PNG / JPG / WebP 画像をオーバーレイできます。
  2. 2
    画像をドロップ or ファイル選択
    最大5枚まで保存。スロットをクリックすると使用する画像を切り替えられます。
オーバーレイ操作パネル
  1. 1
    不透明度
    スライダーでオーバーレイの透過度を調整します。
  2. 2
    スケールモード
    実寸 (1:1) / 幅に合わせる / 高さに合わせる の3種類。
  3. 3
    オフセット (px)
    X / Y 方向のずれを数値入力または矢印ボタンで調整。
    「✥ ドラッグ移動」をONにするとオーバーレイを直接ドラッグで動かせます。
    Shift + 矢印ボタンで 10px 単位移動。
  4. 4
    スクロールに追従
    ONにするとページスクロールと一緒にオーバーレイが動きます。
  5. 5
    差分ハイライト
    ブレンドモード difference でデザインと実装のズレを強調表示します。
キーボードショートカット
  • オーバーレイ 表示 / 非表示 Alt + Shift + D
Tips
  • Figma URLは node-id パラメータ付きのリンクをコピーしてください（フレームを右クリック → リンクをコピー）。
  • 取得画質を 2× にすると Retina ディスプレイでもきれいに表示されます。
  • パネルはヘッダー部分をドラッグして移動できます。
  • 設定（不透明度・オフセット等）はURLごとに自動保存されます。
よくある質問
オーバーレイが表示されない

以下を順番に確認してください。

  • ポップアップの「ON — クリックで非表示」ボタンがON状態になっているか
  • Figma URLに ?node-id= パラメータが含まれているか（フレームを選択した状態でURLをコピー）
  • 取得画質を変更して再読み込みしてみる
  • ページを再読み込みしてから再度「オーバーレイを読み込む」をクリックする
「無効なFigma URLです」と表示される

Figmaでフレームやコンポーネントを選択した状態でURLをコピーしてください。ブラウザのアドレスバーから直接コピーすると node-id が含まれない場合があります。

正しいURL例: https://www.figma.com/design/XXXX/name?node-id=1-2

「トークンが無効です」と表示される

Figma PATの有効期限切れ、または誤ったトークンの可能性があります。ヘッダーの設定アイコン（⚙）から新しいトークンを再入力してください。トークンは Figma → アカウント設定 → Personal access tokens で再発行できます。

PATとは何か、どこで取得できるか

Personal Access Token の略で、Figma APIへのアクセスに使う認証情報です。

取得手順: Figmaにログイン → 左上のアイコン → Settings → Personal access tokens → Generate new token

発行直後にしか全文表示されないため、すぐコピーして保存してください。

ページを移動するたびにオーバーレイが消える

仕様です。ページ遷移（URL変更）のたびにオーバーレイはリセットされます。再度ポップアップから「オーバーレイを読み込む」をクリックしてください。設定（不透明度・オフセット等）はURLごとに自動保存されているため、再読み込み時に引き継がれます。

デザインと実装の位置がずれる

オフセット (px) で X / Y を調整するか、「✥ ドラッグ移動」でオーバーレイを直接動かしてください。スケールモードを「幅に合わせる」にするとレスポンシブページに合わせやすくなります。

特定のサイトでパネルが表示されない / 動かない

一部のサイトでは Content Security Policy (CSP) により拡張機能の動作が制限されることがあります。また、Chrome の内部ページ（chrome://）や Chrome Web Store では拡張機能は動作しません。

パネルが画面外に出て操作できなくなった

拡張機能のアイコンをクリックしてポップアップを開き、「オーバーレイを読み込む」を再度実行するとパネルが初期位置にリセットされます。

画像アップロードの保存上限を超えたい

現在は最大5スロットです。不要な画像のスロットを削除してから新しい画像を追加してください。

トークンをブラウザ終了後も保持したくない

セットアップ画面の「ブラウザ終了後も保存する」のチェックを外すと、セッション（タブを閉じるまで）のみの保持になります。設定を変更するにはヘッダーの設定アイコン（⚙）から再設定してください。
