# D2W Diff

Figma のデザインカンプをブラウザ上のページに半透明でオーバーレイ表示し、デザインと実装の差分を目視確認できる Chrome 拡張機能です。

**バージョン:** 1.1.0  
**対応言語:** 日本語のみ

---

## 特徴

- Figma URL を貼り付けるだけで最新デザインをオーバーレイ表示
- Figma URL を最大3件登録してワンクリックで切替（サムネイル付き）
- PNG / JPG / WebP の画像アップロードにも対応（最大3件・グリッド / リスト表示）
- 差分ハイライトモードでピクセルレベルのズレを強調表示
- 不透明度・オフセット・スケールモードを自由に調整
- Shadow DOM による完全分離でページのスタイルに干渉しない
- キーボードショートカット（Option+D / Alt+D）対応

---

## インストール方法

Chrome ウェブストアから直接インストールできます。

[D2W Diff — Chrome ウェブストア](https://chromewebstore.google.com/detail/d2w-diff/onhbinhmbmojhmnmchbjeocgbclicmmb)

1. 上記リンクを開く
2. 「**Chrome に追加**」をクリック
3. ツールバーに **D2W Diff** のアイコンが表示されれば完了

## 使い方

### 1. Figma API モード

1. ツールバーの D2W Diff アイコンをクリック
2. 初回はセットアップ画面で Figma Personal Access Token（PAT）を入力して保存
3. メイン画面の「Figma API」タブで Figma URL を貼り付け、フロッピーアイコンで保存（最大3件）
4. 保存済み URL のタイルをクリックして切替。読み込み後はサムネイルが表示される
5. 取得画質・スケールモードを選択して「オーバーレイを読み込む」をクリック

> Figma PAT の取得方法：figma.com にログイン → Settings → Security → Personal access tokens → Generate new token

### 2. 画像アップロードモード

1. ツールバーの D2W Diff アイコンをクリック
2. 「画像アップロード」タブをクリック
3. PNG / JPG / WebP をドラッグ＆ドロップ、またはファイルを選択（最大3件保存）
4. 右上のグリッド / リストアイコンで表示切替（リスト表示ではサムネイルとファイル名を表示）
5. タイルをクリックして使用する画像を選択し、「オーバーレイを読み込む」をクリック

### 3. オーバーレイパネルの操作

オーバーレイ読み込み後、ページ右上にコントロールパネルが表示されます。

| 操作 | 方法 |
|------|------|
| 表示 / 非表示 | ON/OFFトグル または Option+D（Mac）/ Alt+D（Win） |
| 不透明度調整 | スライダーで 0〜100% |
| 位置調整 | X/Y オフセットの数値入力または矢印ボタン（Shift+クリックで10px移動） |
| スケール変更 | 実寸(1:1) / 幅に合わせる / 高さに合わせる |
| 差分ハイライト | 「差分ハイライトモード」チェックボックスをオン |
| パネル移動 | ヘッダーをドラッグ |
| 折りたたみ | ヘッダーの「−」ボタン |
| 閉じる | ヘッダーの「×」ボタン |

### 4. キーボードショートカット

| キー | 動作 |
|------|------|
| Option + D（Mac）/ Alt + D（Win） | オーバーレイの表示・非表示をトグル |
